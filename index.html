REVELI-OS

Onde a mágica roda em comandos (e sem anúncios falsos).

PASSO 0: Pegue seu "RG do PC"

Copie o código abaixo, salve como Scanner.bat no Bloco de Notas e execute como Administrador.

@echo off title REVELI-OS Scanner echo Escaneando... Aguarde. set "out=Meu_PC_REVELIOS.txt" wmic computersystem get TotalPhysicalMemory /value > %out% wmic cpu get Name /value >> %out% wmic path win32_VideoController get Name /value >> %out% echo Scan concluido! O arquivo %out% foi criado. pause

PASSO 1: Analisar e Revelar

Abra o arquivo Meu_PC_REVELIOS.txt, copie tudo e cole aqui:

PASSO 2: Aplicar o Boost

Limpeza de DNS, arquivos temporários e liberação de CPU.

@echo off :: REVELI-OS BOOST ipconfig /flushdns del /q /s /f %temp%\* reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\BackgroundAccessApplications" /v "GlobalUserDisabled" /t REG_DWORD /d 1 /f echo Otimizado pelo REVELI-OS! pause

PASSO 3: Botão de Emergência (Reset)

Volta as configurações ao padrão original do Windows.

@echo off :: REVELI-OS RESET reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\BackgroundAccessApplications" /v "GlobalUserDisabled" /t REG_DWORD /d 0 /f netsh winsock reset echo Restaurado pelo REVELI-OS! pause
